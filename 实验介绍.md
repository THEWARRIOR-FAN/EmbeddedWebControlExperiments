# 网页实验文件夹介绍

## 概述

本文件夹用于存放基于网页技术实现的各种控制系统模拟实验，旨在通过交互式网页界面帮助理解和掌握不同控制模型的工作原理和参数调节方法。

## 当前实验

### PID调参实验

**文件：** `PID调参实验.html`

**实验目的：** 直观理解PID控制器的工作原理，掌握PID参数（Kp、Ki、Kd）对系统性能的影响。

**功能特点：**
- 实时水位模拟与可视化显示
- 可调节的目标水位设置
- 交互式PID参数调节（Kp、Ki、Kd）
- 实时曲线显示系统响应
- 理论稳态误差计算
- 系统状态实时监测（当前水位、阀门开度、误差）

**使用方法：**
1. 打开 `PID调参实验.html` 文件
2. 调节右侧参数滑块：
   - 目标水位：设置期望的水位高度
   - Kp（比例）：调节比例增益
   - Ki（积分）：调节积分增益
   - Kd（微分）：调节微分增益
3. 观察左侧水池状态变化和实时曲线
4. 点击画布可暂停/继续实验
5. 暂停状态下可拖动画布查看历史数据
6. 点击「重置」按钮可重置系统状态

**PID参数影响：**
- 纯P控制：系统稳定但存在静差
- P值过大：系统会出现超调和震荡
- 增加I：消除静差，使系统逐渐达到目标值
- 增加D：抑制震荡，使系统响应更平稳

## 技术实现

所有实验均采用纯前端技术实现：
- HTML5 结构
- CSS3 样式

无需后端服务器，可直接在浏览器中打开运行。
