# 网页实验文件夹介绍

## 概述

本文件夹用于存放基于网页技术实现的各类控制系统仿真实验，旨在通过交互式网页界面帮助理解和掌握不同控制模型的工作原理和参数调节方法。

## 当前实验

### PID 调参实验

**文件**：`pid-tuner.html`

**实验目的**：直观理解 PID 控制器的工作原理，掌握 PID 参数（Kp、Ki、Kd）对系统性能的影响。

**功能特点**：
- 实时水位模拟与可视化显示
- 可调节的目标水位设置
- 交互式 PID 参数调节（Kp、Ki、Kd）
- 实时曲线显示系统响应
- 理论稳态误差计算
- 系统状态实时监测（当前水位、阀门开度、误差）

**使用方法**：
1. 打开 `pid-tuner.html` 文件
2. 调节右侧参数滑块：
   - 目标水位：设置期望的水位高度
   - Kp（比例）：调整比例增益
   - Ki（积分）：调整积分增益
   - Kd（微分）：调整微分增益
3. 观察左侧水池状态变化和实时曲线
4. 点击画布可暂停，继续实验
5. 暂停状态下可拖动画布查看历史数据
6. 点击“重置”按钮可重置系统状态

**PID 参数影响**：
- 纯 P 控制：系统稳定但存在静差
- P 值过大：系统会出现超调和振荡
- 增加 I：消除静差，使系统逐渐达到目标值
- 增加 D：抑制振荡，使系统响应更平稳

## 技术实现

所有实验均采用纯前端技术实现：
- HTML5 结构
- CSS3 样式
- JavaScript 逻辑
- Canvas API 数据可视化

无需后端服务器，可直接在浏览器中打开运行。

## 实验价值

通过这些网页实验，可以：
- 直观理解各种控制算法的工作原理
- 掌握参数调节对系统性能的影响
- 比较不同控制策略的优缺点
- 培养控制系统设计和调试能力
- 为实际工程应用提供参考
